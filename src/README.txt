SNAKE GAME


This project is created like simple game written in pure pascal.
Early code is written only in SLOVAKIA language, but it is nearly translated to ENGLISH.

This game shows, how to simply use pascal to create a game.

There is some undone code. Mainly section with AI and translation to English.
AI:
O=off
1=as fast as possible
2=simulation in user selected speed
3=faster than maximum user simulation
4=fast simulation
5=very fast simulation
